This is a new component that is not currently supported as part of the NHS design system. We are providing this to support the ongoing development work on dynamic SEDs.

It has not been tested with users and may change as we learn more about how users will use this component in context.

Provide hint text for selecting multiple options

A limitation of this approach is that it requires users to know to press command on apple or ctrl on windows to select multiple options.

We should provide this information as hint text, indicating the correct key to use based on the user's operating system.
