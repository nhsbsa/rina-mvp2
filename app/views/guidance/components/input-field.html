SED implementation guidance

We recommend following the NHS design system for this component. Below are additional points to note when implementing it as part of dynamic SEDs.

Make text inputs the right size

For discussion NHS guidance states that text inputs should be the right size for the information that the user will put into them - we need to establish sensible defaults that allow for the right size inputs to be generated as part of dynamic SEDs.
