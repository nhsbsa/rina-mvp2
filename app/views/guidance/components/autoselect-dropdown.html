SED implementation guidance

This pattern is specific to EESSI RINA and involves an:

  • autocompletion dropdown
  • autopopulated text input
  • advanced search feature

Given we are considering automation of these forms in their entirety it is important to understand in what contexts users would be interacting with these fields, if at all, before designing a new pattern.

For the purposes of creating PoC dynamic SEDs, we recommend:

  • implementing the autocomplete textbox for the selection field
  • making the autopopulated text input readonly

We will provide styling as part of completing this guidance for readonly fields. We will also review the full functionality for this pattern as part of an applicable user journey in future UCD work.
