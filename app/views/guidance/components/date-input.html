SED implementation guidance

We recommend following the NHS design system for this component. Below are additional points to note when implementing it as part of dynamic SEDs.

Do not use a date picker

For known dates, NHS and GOV.UK design guidance recommends using the date input component rather than a date picker. As SED users will only be entering known dates, a date picker is not required.

Do not include hint text

We recommend not including grey hint text on date inputs as user research has shown that our users can confidently navigate screens without it.
