SED design guidance

The design guide supports the RINA team to build consistent, accessible SEDs.

This is a living document which will expect to evolve as part of ongoing UCD work

Most of the recommended components are based on the NHS Design System.

Some content recommendations are based on the GOV.UK Design System.

The guide provides live examples of the NHS-style SED components and also the code for these components.
