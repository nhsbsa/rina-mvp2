Example SEDs - S044

This is an example SED created using the SED design guidance.

This example is a work in progress. It has not been evaluated as part of an end-to-end user journey where automation may be used to reduce the amount of manual entry required by users.

We have provided working examples of some components that do not exist in the NHS design system, but these may change based on future user testing.

The components that are still work in progress are:

  • autocomplete textbox
  • autoselect dropdown
  • multiselect dropdown

The following features have not been implemented in this example:

  • error validation
  • adding new fieldsets
  • autocomplete textbox behaviour
